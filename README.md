В зоне работают:
Если зона test.ru
* - любые поддомены этой зоны test.ru
*.info - любые поддомены для *.info.test.ru
*.sub.admin - любая вложенность субдоменов для *.sub.admin.test.ru
sub.admin - точное совпадение записей sub.admin.test.ru


Как называется зона, такой ORIGIN - корневой домен и будет парсится и подставляться к занчениям внутри зоны.
